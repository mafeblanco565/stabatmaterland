# Stabat Mater - Landing Page

Landing page completa para la propuesta performática del Stabat Mater de Pergolesi 2026.

## Contenido

- `index.html` - Página web completa y autocontenida con todos los estilos y scripts incluidos

## Características

- Diseño responsive (móvil, tablet, escritorio)
- Efectos parallax en el hero
- Animaciones suaves al hacer scroll
- Videos embebidos de YouTube
- Botón flotante de WhatsApp
- Navegación suave entre secciones
- Texturas barrocas sutiles en el fondo
- Tipografía Playfair Display y Inter desde Google Fonts

## Cómo usar

1. Abrir `index.html` en cualquier navegador web
2. Todos los estilos están incluidos en el archivo HTML
3. Las imágenes están referenciadas desde URLs de Vercel Blob Storage
4. No requiere ninguna dependencia externa excepto las fuentes de Google Fonts

## Secciones

1. **Hero** - Imagen de la Pietà con título principal
2. **Descripción** - Información sobre la propuesta artística
3. **Formato Artístico** - Tres tarjetas dinámicas con información de instrumentos, voces y bailarines
4. **Solistas** - Biografías completas de Mayra Vargas y Nicolás Vera-Picón con fotos
5. **Videos** - Videos embebidos de YouTube de ambos solistas
6. **Contacto** - Email y WhatsApp con fondo de partitura musical

## Compatibilidad

Compatible con todos los navegadores modernos (Chrome, Firefox, Safari, Edge)
